# Amadeus Cafe – Website Redesign (Version 1)

This repository contains Version 1 of the Amadeus Cafe website redesign created for Project A.  
This version was built using a Microsoft AI-assisted workflow.

## Project Overview
The goal of this project was to redesign an existing local business website as a static prototype.  
This version focuses on creating the initial structure, layout, and base styling of the site.

## Technologies Used
- HTML
- CSS
- JavaScript (minimal, if applicable)
- Visual Studio Code
- Microsoft AI tools (ChatGPT / GitHub Copilot)

## How to Run the Project
1. Clone or download this repository.
2. Open the `index.html` file in a web browser.
   - Alternatively, open the folder in VS Code and use Live Server extension.

## Project Structure
- `index.html` – Home page
- `about.html` – About the business
- `menu.html` – Menu overview
- `contact.html` – Contact information and hours
- `css/` – Stylesheets
- `images/` – Image assets 

## Notes
This is a static prototype created for educational purposes and is not a production-ready website.
